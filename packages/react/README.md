## Install

```bash
yarn install @livekit/components-react
```

## Usage

```tsx
import { LiveKitRoom } from '@livekit/components-react';

<LiveKitRoom token="<livekit-token>" serverUrl="<url-to-livekit-server>" connect={true} />;
```
